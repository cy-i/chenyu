## 欲买桂花同载酒，终不似，少年游
>大家好像都在说 “欲买桂花同载酒，终不似，少年游”，好像每一个人都很怀念过去。可是诸君，还有一句话叫做 “莫道桑榆晚，为霞尚满天”。人生还很长，一程有一程的风景。二十岁没有完成的事情，三十岁去完成也没有关系啊。就算是朝花夕拾，你觉得没有意义了，那也无所谓啊。人生于天地间，若白驹过隙，忽然而已。所以，岁月不待人，及时当勉励，重在的是诗酒趁年华。
